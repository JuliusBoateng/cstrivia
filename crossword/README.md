# Crossword App

This directory contains the Django application that powers CS Trivia.

It includes the core backend logic for puzzles and design notes.

## Structure

models.py  
Defines database models such as `Board`, `Clue`, and `DesignNote`.

views/  
Contains the views used to render puzzle pages, the homepage, and design notes.

templates/crossword/  
Django HTML templates.

static/crossword/  
CSS, images, icons, and compiled frontend bundles.

urls.py  
Routes for the crossword application.
